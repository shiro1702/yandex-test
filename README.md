# Клуб четырех коней — лендинг

Адаптивный одностраничный лендинг по макету (тестовое задание): «Международный васюкинский турнир», блоки поддержки, программы, этапов развития и участников.

## Демо

**Опубликованная версия:** [https://shiro1702.github.io/yandex-test/](https://shiro1702.github.io/yandex-test/)

## О проекте

Верстка с упором на соответствие макету и аккуратное поведение на разных ширинах экрана: без горизонтального скролла, читаемые переносы, отсутствие наложения текста на декоративные элементы. Контент для мобильной и десктопной версий не дублируется отдельными копиями блоков — адаптация через разметку и стили.

## Соответствие требованиям ТЗ

**Из задания:** макет Figma; **HTML + CSS + JS**; Pixel Perfect; адаптив (без горизонтального скролла и «ломаных» текстов); без дублирования одного и того же текста отдельно для мобилки и десктопа; бегущая строка; якоря с первого экрана; карусель участников с **loop** и автосменой **каждые 4 с**; карусель этапов **без** loop и автосмены; дополнительная анимация по усмотрению.

**В репозитории:** см. таблицу ниже. Слайдеры собраны на **Swiper 11** (CDN); остальная логика — нативный JS в `script.js`.

| Критерий | Реализация |
|----------|------------|
| Pixel Perfect, адаптив | Сетка и типографика из макета, брейкпоинты в CSS |
| Бегущая строка | Блок `.ticker` (CSS-анимация), в шапке и подвале |
| Кнопки первого экрана | Якоря `#support` и `#about` |
| Карусель участников | Swiper: **loop**, **autoplay 4 с**, стрелки, счётчик «текущий / всего», 1 или 3 карточки по ширине экрана |
| Карусель этапов | Swiper только на мобильных; без автопрокрутки и зацикливания |
| Анимация | Появление блоков при скролле (`IntersectionObserver`), параллакс в hero, учёт `prefers-reduced-motion` |

## Технологии

- HTML5, CSS3 (в т.ч. анимации и кастомные свойства)
- JavaScript без сборщика
- [Swiper 11](https://swiperjs.com/) (подключение с CDN: стили + скрипт) — слайдеры этапов и участников
- Шрифты: [Google Fonts](https://fonts.google.com/) — Golos Text, Merriweather

## Структура репозитория

```
├── index.html      # разметка страницы
├── styles.css      # стили
├── script.js       # слайдеры, reveal, адаптив инициализации этапов
├── storage/        # изображения и SVG по секциям
└── README.md
```

## Локальный запуск

Статическая страница: достаточно открыть `index.html` в браузере или поднять любой статический сервер из корня проекта, например:

```bash
npx --yes serve .
```

Затем перейти по адресу, который выведет команда (обычно `http://localhost:3000`).
